Turmeric Business Website

A production-ready static website built for a turmeric-based business to showcase products, brand story, and contact information.

Overview

This project is a responsive and lightweight website developed using HTML, CSS, and JavaScript. It is designed to provide a fast, user-friendly experience while effectively presenting the client’s turmeric products and business details.

Features
Fully responsive design (mobile, tablet, desktop)
Clean and modern UI
Fast loading static pages
Product showcase section
Contact section for customer inquiries
SEO-friendly structure
Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
